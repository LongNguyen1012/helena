Helena & Ringer
Web Automation for End Users

Helena is a programming language with a programming-by-demonstration tool for automating web interactions and web scraping. With Helena's Chrome extension, demonstrate how to scrape the first row of data, then get a script for the remaining rows.
Tools
Helena is a programming language for editing, extending, and understanding web automation programs written using Helena's programming-by-demonstration tool, Rousillon.

GitHub | Publications

Rousillon is a browser extension for writing programs in the Helena language using programming by demonstration techniques. Using the extension, users can demonstrate how to scrape the first row of a multi-relational dataset and receive a generalized Helena program for scraping remaining data.

GitHub | Video Tutorials | Tips & Tricks

Ringer is a browser record and replay tool. Ringer takes as input a user demonstration of a web interaction and produces as output a robust script for repeating the same interaction on the same webpages, even as those pages change. Helena uses the Ringer library to record user interactions.

Projects
Housing Markets We are partnering with a team of sociologists to study the practices and attitudes of rental Property Owners/Managers (POMs) in a tight housing market, scraping rental listings to learn how POM practices vary across neighborhoods.
Hurricane Harvey We are collaborating with a small team of transportation engineers to study the effects of Hurricane Harvey by collecting a wide variety of datasets, from rescue request listings to power outage details.
Charitable Foundation Communications We are providing data to a group of public policy researchers studying how charitable foundations communicate with their supporters, collecting data from a variety of social media and and community foundation sites.
And More! We have also been collecting data with collaborators from backgrounds ranging from economics to art to environment and forest sciences. Check back as we keep adding details on our collaborations!
